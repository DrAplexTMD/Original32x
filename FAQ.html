Frequently Asked Questions

What is Original 32x?

Original 32x is a Minecraft Resource Pack. This Resource doubles up the normal Minecraft Pixels. It changes the pixels size from 16x16 to 32x32

Is it Free?

Yes it is fully a Open Source project. For Anyone, Anyone from worldwide countrys, It is 100% Free.
